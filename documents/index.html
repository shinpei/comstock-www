Install

$ go get github.com/shinpei/comstock

Setup

$ comstock

Login

$ comstock login

Stock your command 

$ coffee -c -O . hoge.coffee
$ comstock save # save former command to comstock

Fecth your command

$ comstock list
1: coffee -c -0 . hoge.coffee
$ comstock run 1 # or you can use 'comstock get 1' and copy and paste it

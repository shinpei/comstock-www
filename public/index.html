Install

$ go get github.com/shinpei/comstock
Or, from homebrew (Mac only) 
$ brew tap shinpei/comstock
$ brew install comstock

Login

$ comstock login

Stock your command 

$ git diff HEAD^ --name-only
$ comstock save # save former command to comstock

Fecth your command

$ comstock list
1: git diff HEAD^ --name-only
$ comstock get 1

Run your command 

$ comstock list
1: git diff HEAD^ --name-only
$ comstock run 1
